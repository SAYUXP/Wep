Política de Privacidad

Última actualización: 2026

Este sitio web ofrece un juego educativo de preguntas y respuestas. Al utilizar el sitio aceptas las prácticas descritas en esta política de privacidad.

Información que recopilamos

Este sitio puede almacenar la siguiente información:

  • Nombre de usuario elegido por el jugador
  • Puntaje obtenido en el juego
  • Categoría del juego
  • Fecha o estadísticas del juego

No recopilamos datos personales sensibles como correo electrónico, dirección o información de pago.

Uso de la información

La información se utiliza únicamente para:

  • Mostrar rankings de jugadores
  • Guardar resultados del juego
  • Mejorar la experiencia del usuario

Almacenamiento de datos

Los datos se almacenan utilizando servicios de base de datos proporcionados por Firebase (Google).

Información pública

El nombre de usuario y el puntaje pueden aparecer públicamente en el ranking del juego.

Seguridad

Se aplican medidas razonables para proteger los datos almacenados, aunque ningún sistema en internet puede garantizar seguridad absoluta.

Cambios en esta política

Esta política puede actualizarse en cualquier momento para mejorar la transparencia del servicio.

Contacto

Si tienes preguntas sobre esta política, puedes contactarnos a través de la seccion Contacto.
